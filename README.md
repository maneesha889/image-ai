# AI Image Description Application

This is a Spring Boot AI web application that **describes an image uploaded by the user**.

The user uploads an image from their system, and the application sends the image to an AI model (OpenAI).  
The AI looks at the image and returns a **simple text description**, which is then shown on the web page.

---

## Features
- Upload an image (JPG/PNG)
- AI-generated description of the image
- Simple and clean web UI
- Built using Spring Boot and Java
- Can be integrated with JSP / HTML / CSS / JavaScript for the frontend

---

##  Technologies Used

- **Backend:** Spring Boot, Java
- **AI:** OpenAI (Vision / Image-to-text)
- **Frontend:** JSP (or HTML/CSS/JavaScript)
- **Build Tool:** Maven
